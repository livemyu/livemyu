## Huang Yu · 黄昱

中山大学毕业 · 做两件跟声音有关的事 —— 让声音 **看得见**，让语音 **被听懂**。

**看得见的声音** → [audioscope](https://github.com/livemyu/audioscope)
音频可视化工具包：波形、STFT 谱、梅尔谱、色度、声谱图……交互式浏览与导出，面向音频分析与教学。

**听得懂的语音** → [voxlink](https://github.com/livemyu/voxlink)
语音大模型工具包：把语音编码器 / 离散语音 token 接入大语言模型，做语音理解、语音问答与语音指令跟随。

NumPy / SciPy 内核，无 GPU、无网络也能跑核心与单测。离线优先，CI 全绿。
